# Discrete Mathematics Notes

This repository contains structured lecture notes and examples covering core topics in Discrete Mathematics. The material is intended to support students in understanding fundamental concepts in logic, set theory, proofs, and number theory.

## Contents

The notes include coverage of the following topics:

- Set Theory and Set Operations  
- Propositional and Predicate Logic  
- Truth Tables and Logical Equivalences  
- Proof Techniques (Direct, Contradiction, Induction)  
- Quantifiers and Arguments  
- Elementary Number Theory  
- Divisibility and Prime Factorization  
- Mathematical Reasoning and Problem Solving

## Purpose

This project serves as a personal academic resource for reviewing and organizing Discrete Mathematics material. It may also be useful for other students seeking clear explanations and worked examples.

## Usage

You are welcome to view, download, and study these notes for educational purposes. If you use or reference this material, please provide appropriate credit.
